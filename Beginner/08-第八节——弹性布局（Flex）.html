一、概念

弹性布局（Flex）提供更加有效的方式对容器中的子元素进行排列、对齐和分配剩余空间。容器默认存在主轴与交叉轴，子元素默认沿主轴排列，子元素在主轴方向的尺寸称为主轴尺寸，在交叉轴方向的尺寸称为交叉轴尺寸。弹性布局在开发场景中用例特别多，比如页面头部导航栏的均匀分布、页面框架的搭建、多行数据的排列等等。和前端中的flex布局几乎一致，属性也一致。

二、布局方向

枚举值说明

在弹性布局中，容器的子元素可以按照任意方向排列。通过设置参数direction，可以决定主轴的方向，从而控制子组件的排列方向。文档地址：https://developer.harmonyos.com/cn/docs/documentation/doc-references-V3/ts-appendix-enums-0000001478061741-V3#ZH-CN_TOPIC_0000001574248789__flexdirection

代码

@Entry
@Component
struct StackPage {
  build() {
    /**
     * 使用FlexDirection对应的枚举值设置布局方向
     * 如 横排
     */
    Flex({ direction: FlexDirection.Row }) {
      Column() {

      }.width(100).height(100).backgroundColor(Color.Blue)

      Column() {

      }.width(150).height(150).backgroundColor(Color.Pink).zIndex(2)

      Column() {

      }.width(200).height(200).backgroundColor(Color.Grey)
    }.margin({ top: 100 }).width(350).height(350).backgroundColor(0xe0e0e0)
  }
}

三、布局换行

枚举值说明

弹性布局分为单行布局和多行布局。默认情况下，Flex容器中的子元素都排在一条线（又称“轴线”）上。wrap属性控制当子元素主轴尺寸之和大于容器主轴尺寸时，Flex是单行布局还是多行布局。在多行布局时，通过交叉轴方向，确认新行堆叠方向。

代码

@Entry
@Component
struct StackPage {
  build() {
    /**
     * 使用FlexWrap对应的枚举值设置是否换行 默认不换行
     * 如 设置换行
     */
    Flex({ direction: FlexDirection.Row, wrap: FlexWrap.Wrap }) {
      Column() {

      }.width(100).height(100).backgroundColor(Color.Blue)

      Column() {

      }.width(150).height(150).backgroundColor(Color.Pink).zIndex(2)

      Column() {

      }.width(200).height(200).backgroundColor(Color.Grey)
    }.margin({ top: 100 }).width(350).height(350).backgroundColor(0xe0e0e0)
  }
}

四、主轴对齐方式（横排）

通过justifyContent参数设置在主轴方向的对齐方式。

FlexAlign.Start

子组件在主轴方向起始端对齐， 第一个子组件与父元素边沿对齐，其他元素与前一个元素对齐。

FlexAlign.Center

子组件在主轴方向居中对齐

FlexAlign.End

子组件在主轴方向终点端对齐, 最后一个子组件与父元素边沿对齐，其他元素与后一个元素对齐。

FlexAlign.SpaceBetween

Flex主轴方向均匀分配弹性元素，相邻子组件之间距离相同。第一个子组件和最后一个子组件与父元素边沿对齐

FlexAlign.SpaceAround

Flex主轴方向均匀分配弹性元素，相邻子组件之间距离相同。第一个子组件到主轴起始端的距离和最后一个子组件到主轴终点端的距离是相邻元素之间距离的一半

FlexAlign.SpaceEvenly

Flex主轴方向元素等间距布局，相邻子组件之间的间距、第一个子组件与主轴起始端的间距、最后一个子组件到主轴终点端的间距均相等

代码

@Entry
@Component
struct StackPage {
  build() {
    Flex({ direction: FlexDirection.Row, justifyContent: FlexAlign.Start }) {
      Column() {

      }.width(20).height(100).backgroundColor(Color.Blue)

      Column() {

      }.width(30).height(150).backgroundColor(Color.Pink).zIndex(2)

      Column() {

      }.width(40).height(200).backgroundColor(Color.Grey)
    }.margin({ top: 100 }).width(350).height(350).backgroundColor(0xe0e0e0)
  }
}

五、交叉轴对齐方式（竖排）

可以通过Flex组件的alignItems参数设置子组件在交叉轴的对齐方式。容器和子元素都可以设置交叉轴对齐方式，且子元素设置的对齐方式优先级较高。

ItemAlign.Auto

使用Flex容器中默认配置

ItemAlign.Start

交叉轴方向首部对齐

ItemAlign.Center

交叉轴方向居中对齐

ItemAlign.End

交叉轴方向底部对齐

ItemAlign.Stretch

交叉轴方向拉伸填充，在未设置尺寸时，拉伸到容器尺寸

代码

@Entry
@Component
struct StackPage {
  build() {
    Flex({ direction: FlexDirection.Row, justifyContent: FlexAlign.Center, alignItems: ItemAlign.Center }) {
      Column() {

      }.width(20).height(100).backgroundColor(Color.Blue)

      Column() {

      }.width(30).height(150).backgroundColor(Color.Pink).zIndex(2)

      Column() {

      }.width(40).height(200).backgroundColor(Color.Grey)
    }.margin({ top: 100 }).width(350).height(350).backgroundColor(0xe0e0e0)
  }
}

六、自适应拉伸

在弹性布局父组件尺寸不够大的时候，通过子组件的下面几个属性设置其在父容器的占比，达到自适应布局能力。

flexBasis

设置子组件在父容器主轴方向上的基准尺寸。如果设置了该值，则子项占用的空间为设置的值；如果没设置该属性，那子项的空间为width/height的值

@Entry
@Component
struct StackPage {
  build() {
    Flex() {
      // Text('flexBasis("auto")')
      //   .flexBasis('auto') // 未设置width以及flexBasis值为auto，内容自身宽松
      //   .height(100)
      //   .backgroundColor(0xF5DEB3)
      // Text('flexBasis("auto")' + ' width("80%")')
      //   .width('80%')
      //   .flexBasis('auto') //设置width以及flexBasis值auto，使用width的值
      //   .height(100)
      //   .backgroundColor(0xD2B48C)
    //
    //   Column() {
    //     Text('flexBasis(100)') // 未设置width以及flexBasis值为100，宽度为100vp
    //       .fontSize(15)
    //       .flexBasis(100)
    //       .height(100)
    //       .backgroundColor(0xF5DEB3)
    //     Text('和我一样长')
    //     .width(100)
    //     .height(100)
    //     .backgroundColor(0xF5DEB3)
    //     .margin({top: 20})
    //   }

    //
      Text('flexBasis(100)')
        .fontSize(15)
        .flexBasis(100)
        .width(300) // flexBasis值为100，覆盖width的设置值，宽度为100vp，也就是说flexBasis如果有具体值那么会直接进行覆盖
        .height(100)
        .backgroundColor(0xD2B48C)
    }.width('90%').height(120).padding(10).backgroundColor(0xAFEEEE)
  }
}

flexGrow

设置父容器的剩余空间分配给此属性所在组件的比例。用于“瓜分”父组件的剩余空间。

@Entry
@Component
struct StackPage {
  build() {
    Flex() {
      Text('占领剩余部分')
        .flexGrow(1)
        .height(100)
        .backgroundColor(0xF5DEB3)

      Text('固定宽度')
        .width(100)
        .height(100)
        .backgroundColor(0xD2B48C)
      Text('固定宽度')
        .width(50)
        .height(100)
        .backgroundColor(Color.Red)

    }.width('100%').height(120).backgroundColor(0xAFEEEE)
  }
}
